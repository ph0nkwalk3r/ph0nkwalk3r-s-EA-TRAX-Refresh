# ph0nkwalk3r’s EA TRAX™ Refresh

A modern EA TRAX soundtrack refresh for **Need for Speed: Most Wanted (2005)**.

This project replaces the original EA TRAX music metadata with a curated modern playlist while preserving the aggressive, high-energy identity of classic EA soundtracks.

## Features
- Modern electronic, metal, hip-hop, and global tracks
- Music-only mod (metadata changes only)
- No gameplay, AI, pursuit, or physics edits
- Compatible with Redux and Extra Options

## Installation
1. Backup your original `SOUND` and `PFDATA` folders
2. Import the included `.nfsms` file using NFS-VltEd
3. Copy the `SOUND` folder into your game directory

## Backup
Before installing, create a vault backup using **NFS-VltEd → File → Export → All**.

## Notes
- Audio files are provided for modding purposes only
- Original game required
